## C
给一些棋子，若一个棋子只经过行变换或者列变换就能到达另一个棋子，这些能到达的棋子称为一个set，
若 a可以到达b，b可到达c，那么a，b，c 也是一个set。
给n个棋子并且标号（1～n），并给出m对编号，要求这些对的棋子类型出现在一个set，
并且每两个相同类型的棋子一定属于同一个set。
把每一行（平行）作为只属于特定类型，比如第一行只放类型为1的棋子，
然后对于第i对，就把它放在第i列上，两个元素各自在所在行。
最后随便找个地方安排那些孤单的点。
